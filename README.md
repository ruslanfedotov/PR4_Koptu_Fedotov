## Архитектура приложения

Десктопное приложение на WPF с навигацией между тремя страницами для расчёта математических выражений по варианту 8.

### Навигация
- Кнопки в верхней части главного окна для переключения между страницами
- При запуске открывается Страница 1

### Страница 1
- Поля ввода: x, y, z
- Кнопки: «Вычислить», «Очистить»
- Вывод результата в текстовое поле
- Проверка: y > 0 (для логарифма), знаменатель ≠ 0

### Страница 2
- Поля ввода: x, m
- Выбор функции f(x): sh(x), x², e^x
- Условное вычисление в зависимости от соотношения x и m
- Кнопки: «Вычислить», «Очистить»

### Страница 3
- Поля ввода: x₀, xₖ, dx
- Построение графика функции
- Многострочный вывод всех значений x и y
- Кнопки: «Построить», «Очистить»
